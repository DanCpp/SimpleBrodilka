The symbols to make(set up) a map:
"@" - That's a player
"#" - That's a wall
"." - That's an empty place

To change map use map.txt (There's should be generator in the near future(maybe))

Keys:
UpArrow - move up
DownArrow - move down
LeftArrow - move left
RightArrow - move right
Escape - exit the app
